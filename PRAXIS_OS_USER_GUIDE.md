# Praxis OS - Desktop Environment Guide

## 🖥️ Welcome to Praxis OS

Praxis OS is a complete operating system-style interface for Agent Zero, providing an intuitive desktop environment for managing your AI agent platform.

---

## 🎯 Quick Start

### Access the Interface

1. **Default OS Interface**: Visit `http://localhost:50080` - automatically redirects to OS desktop
2. **Classic Interface**: Visit `http://localhost:50080/?os=false` for original interface

### First Steps

1. **Desktop Icons**: Click icons to open applications
2. **Taskbar**: Use taskbar at bottom for quick access and system info
3. **Agent Assistant**: Hover over the robot 🤖 in bottom-right for quick chat
4. **Start Menu**: Click the Praxis logo 🪟 for application launcher

## 🪟 Windows System

### Window Controls

Each window has three control buttons:
- 🔴 **Close**: Closes the window completely
- 🟡 **Maximize**: Expands to full screen
- 🟢 **Minimize**: Hides window (still in taskbar)

### Window Management

**Drag Windows**: Click and drag the title bar to move windows
**Resize Windows**: Drag the bottom-right corner to resize
**Bring to Front**: Click any window to bring it to the front
**Taskbar Toggle**: Click taskbar items to show/hide windows

### Multi-Window Support

- Open multiple windows simultaneously
- Windows automatically layer when clicked
- Minimize unused windows to reduce clutter
- Maximize important windows for full focus

## 📊 Email Service Control

### Dashboard Overview

The Email Service window provides real-time monitoring and control:

**Statistics Cards**:
- **Processed**: Total emails processed
- **Pending**: Emails awaiting processing
- **Trades**: Trades executed today
- **Exceptions**: Items requiring human review

**Control Buttons**:
- ▶️ **Start Service**: Begin email processing
- ⏸️ **Stop Service**: Pause email processing
- 🛑 **Emergency Stop**: Immediately halt all operations

### Activity Monitoring

**Last Email**: Shows most recently processed email timestamp
**Queue Status**: Current depth of email queue
**Service Indicator**: Green (active) or red (inactive)

## 🦞 NanoClaw Configuration

### Agent Settings

Configure your AI agent behavior:
- **Model**: LLM model selection
- **Max Tokens**: Context window size
- **Temperature**: Response randomness (0.0-1.0)

### File System

- **Work Directory**: Agent workspace location
- **Memory File**: CLAUDE.md memory file
- **Auto-save**: Automatic memory saving

### Security

- **Sandbox Mode**: Container isolation
- **API Rate Limiting**: Request throttling
- **Audit Logging**: Activity tracking

### Configuration Actions

- 💾 **Save Config**: Apply and save settings
- 🔄 **Restart Agent**: Restart with new config
- 🔄 **Factory Reset**: Reset all settings to defaults

**Minimum**:

- 2 CPU cores
- Modern web browser

### Performance Tips
